# RESTful

REST: Representational State Transfer, 表述性状态转移

RESTful: 满足REST架构风格的接口

  • 以资源为基础: 任何有被引用的必要的事物
  • 统一资源接口: 赋予GET, POST, PUT, DELETE的不同意义
